## fastify 기본 설정

- npm 초기화

```bash
npm init`
```

- `fastify` 설치

```bash
npm i fastify
```

- `typescript`, `@types/node` 설치

@type/node는 Node.js의 타입 정의를 제공하는 TypeScript 패키지

```bash
npm i -D typescript @types/node
```

- typescript 설정 파일 생성

```bash
npx tsc --init
```

- 개발용 패키지 설치

```bash
npm i -D ts-node nodemon tsx
```
